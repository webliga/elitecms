В этой папке находятся файлы отображения каждого модуля.
Тут они переопределяются или создаются новые. Тоесть каждый модуль имеет свою папку templates, 
что б иметь независимый файлы отображения. Но их можно переопределить, просто в папке modules Вашего шаблона
создать папку с именем модуля и скопировать туда нужные файлы. Система автоматом будет в первую очередь искать
файлы отображения в текущем шаблоне, если их нету, тогда берет файлы из своей папки
